Laravel Uptime Monitor Cachet Integration - Documentation

v 1.0

Description

This is an Addon Package for https://github.com/spatie/laravel-uptime-monitor. This Package provides an automatic Cachet Incident Creation when a site went down.

Basic Installation

You can install this package via composer using:

composer require lkdev/laravel-uptime-monitor-cachet-integration

Than you need to register the service provider after the service provider from laravel-uptime-monitor

            
'providers' => [
    // ...
    LKDevelopment\UptimeMonitorCachetIntegration\UptimeMonitorAPIServiceProvider::class,
];

Configuration

To publish the config file to config/laravel-uptime-monitor-cachet-integration.php run: 

php artisan vendor:publish --provider="LKDevelopment\UptimeMonitorCachetIntegration\UptimeMonitorAPIServiceProvider"

In the Config File, you can change your Cachet Endpoint and your Cachet Token. See Cachet Documentation for more Information about Cachet.
